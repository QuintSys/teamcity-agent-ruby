# teamcity-agent-ruby

Dockerfiles for TeamCity Build Agent with pre-installed RVM and Ruby

This image is based on the official JetBrains TeamCity build agent image and extends it by installing RVM, Ruby and bundler.

## Installed software

### Ruby

* ruby2.4.3

### Bundle

* bundler 1.16.1

### NodeJs

* 8.9.4

### Yarn

* 1.3.2

### DB

* PostgreSQL dev lib
